# Mind Lapse - Project Overview

## Vision
A high-signal, premium news community and blog specifically for Gen Z (ages 18-25), focusing on AI news, tech trends, and "highly interesting things" happening worldwide. 

## The Vibe
- **Not Boring:** Moving away from traditional, dry news layouts.
- **Premium & Gen Z Aesthetic:** Attractive, high-value packaging and presentation.
- **Platform:** Starting with a WhatsApp community and a simple, high-end website.

## Reference Channels (The Benchmarks)
We aim to beat these in speed and quality:
1. **Full Disclosure** (YouTube)
2. **Varun Mayya** (YouTube/Reels)
3. **100x Engineer** (YouTube)
4. **AI Search** (YouTube)
5. **Hacker News** (Website - for technical depth and signal)

## Target Audience
- College students.
- Gen Z professionals (up to 25).
- Tech-curious individuals who want to cut through the hype.

## Technical Strategy
- **Hosting:** Starting with Vercel (Free Tier).
- **Analytics:** Integration with Google Analytics (confirmed compatible with `.vercel.app`).
- **Media:** 
    - YouTube: Embedded to save bandwidth and storage.
    - Images: Optimized and stored in the repository or a lightweight CDN.
- **Content Management:** Initially static or using a lightweight CMS for speed.
